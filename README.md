# Surajmal Sons — Interior Design & Furniture Website

A production-ready marketing site built with React + TypeScript + Vite + Tailwind CSS v4,
Framer Motion, React Router, Swiper and React Hook Form.

## Getting started

```bash
npm install
npm run dev       # local dev server
npm run build     # production build → dist/
npm run preview   # preview the production build
```

## Important — before you launch

1. **Logo**: no logo file was supplied with the brief, so `src/components/Logo.tsx`
   currently renders a placeholder wordmark. Replace its contents with your real logo
   (drop the file in `public/logo.svg` or `.png` and swap in an `<img>` tag).
2. **Photography**: every image in the site is an abstract SVG placeholder generated by
   `src/utils/placeholder.ts`, so the layout, spacing and hover states can be reviewed
   without real photos. Replace the `image` fields in `src/data/services.ts` and
   `src/data/projects.ts`, and the `placeholderImage('seed')` calls in `src/sections/*`
   and `src/pages/Gallery.tsx` / `About.tsx`, with real project photography.
3. **Business details**: phone, WhatsApp number, email and address live in
   `src/constants/index.ts` — update these with your real details.
4. **Contact form**: `src/pages/Contact.tsx` currently logs submissions to the console.
   Wire the `onSubmit` handler to your email service, CRM, or backend endpoint.
5. **Map**: the Contact page has a placeholder map panel — swap it for a real Google Maps
   embed (`<iframe>`) once you have your Google Maps embed URL.

## Project structure

```
src/
  assets/        static assets
  components/    shared UI (Navbar, Footer, Button, Cards, Logo, PageHeader…)
  constants/     business info, nav links
  data/          services, projects, testimonials, process, FAQs (typed content)
  hooks/         shared React hooks
  layouts/       MainLayout (navbar + footer wrapper)
  pages/         one file per route
  sections/       composable page sections (Hero, ServicesGrid, ProcessSection…)
  types/         shared TypeScript types
  utils/         helpers (placeholder image generator)
```
